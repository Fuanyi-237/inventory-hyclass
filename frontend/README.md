# Inventory System Frontend

This directory will contain the React-based frontend for the Inventory System.

## Features
- Admin/Superadmin login
- Dashboard with inventory stats and alerts
- Item management (add, edit, sign in/out)
- Category management
- User management
- Weekly report download
- In-app notifications

## Getting Started
1. Install dependencies: `npm install`
2. Start the dev server: `npm start`

---

Backend is in `/backend`.
